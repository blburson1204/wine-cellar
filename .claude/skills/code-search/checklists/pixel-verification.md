# Pixel-Perfect UI Verification Checklist

## Layout Accuracy

- [ ] Spacing matches design (margin, padding)
- [ ] Alignment matches wireframe
- [ ] Grid/flexbox layout correct
- [ ] Element sizes match design
- [ ] Border radius matches design

## Typography

- [ ] Font family matches design
- [ ] Font size matches design
- [ ] Font weight matches design
- [ ] Line height matches design
- [ ] Letter spacing matches design

## Colors

- [ ] Text colors match design tokens
- [ ] Background colors match design tokens
- [ ] Border colors match design tokens
- [ ] Hover/focus colors defined
- [ ] Disabled state colors defined

## Responsive Behavior

- [ ] Mobile breakpoint correct
- [ ] Tablet breakpoint correct
- [ ] Desktop breakpoint correct
- [ ] Elements stack properly on mobile
- [ ] Touch targets 44x44px minimum

## Component States

- [ ] Default state matches design
- [ ] Hover state matches design
- [ ] Focus state matches design
- [ ] Active state matches design
- [ ] Disabled state matches design
- [ ] Loading state matches design
- [ ] Error state matches design

## Visual Regression

- [ ] Screenshots at all breakpoints
- [ ] Diff tolerance < 2% configured
- [ ] Baseline images up to date
- [ ] Critical pages covered

## Design System Compliance

- [ ] Components from project UI components
- [ ] No raw Tailwind colors
- [ ] Design tokens used consistently
- [ ] No arbitrary values
